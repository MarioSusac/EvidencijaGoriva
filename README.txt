Naziv projekta:
Evidencija goriva v1.0

Izradio:
Mario Sušac
(za potrebe predmeta "XML programiranje")

- Projekt predstavlja web stranicu za evidenciju točenja goriva.
- Za pokretanje je potreban "Apache server"
- Projekt se sastoji od 3 dokumenta:
	• forma.php - predstavlja početnu stranicu sa formom za unos podataka
	• baza.xml - xml dokument u koji se zapisiju (dodaju) ili čitaju podaci iz forme
	• citaj_xml.php - stranicu koja učitava podatke iz XML dokumenta u HTML tablicu
